# Tools

Essential development tools for Node.js developers.

## Package Management

- **npm** - Default package manager for Node.js
  - [Website](https://www.npmjs.com/)
  - [Docs](https://docs.npmjs.com/)
  
- **Yarn** - Fast, reliable package manager
  - [Website](https://yarnpkg.com/)
  - [Docs](https://yarnpkg.com/getting-started)

- **pnpm** - Fast, disk space efficient package manager
  - [Website](https://pnpm.js.org/)
  - [Docs](https://pnpm.js.org/en/)

## Development Servers & Hot Reload

- **nodemon** - Monitor changes and auto-restart server
  - Install: `npm install -g nodemon`
  - [GitHub](https://github.com/nodemon/nodemon)

- **supervisor** - Restart server on file changes
  - Install: `npm install -g supervisor`
  - [GitHub](https://github.com/petruisfan/node-supervisor)

## Process Managers

- **PM2** - Advanced Node.js process manager with clustering
  - [Website](https://pm2.io/)
  - [GitHub](https://github.com/Unitech/pm2)
  - Features: Load balancing, monitoring, logging

- **forever** - Simple process manager for running scripts
  - Install: `npm install -g forever`
  - [GitHub](https://github.com/foreversd/forever)

## CLI Development

- **commander** - CLI utility for argument parsing
  - [GitHub](https://github.com/tj/commander.js/)

- **Inquirer** - Interactive CLI prompts
  - [GitHub](https://github.com/SBoudrias/Inquirer.js)

- **chalk** - Terminal string styling
  - [GitHub](https://github.com/chalk/chalk)

- **ora** - Elegant terminal spinner
  - [GitHub](https://github.com/sindresorhus/ora)

## API Development & Testing

- **Postman** - API client for testing endpoints
  - [Website](https://www.postman.com/)
  - [Download](https://www.postman.com/downloads/)

- **Insomnia** - Open-source API client
  - [Website](https://insomnia.rest/)
  - [GitHub](https://github.com/Kong/insomnia)

- **httpie** - Human-friendly CLI HTTP client
  - [Website](https://httpie.io/)
  - [GitHub](https://github.com/httpie/httpie)

## Database Tools

- **MongoDB Compass** - GUI for MongoDB
  - [Website](https://www.mongodb.com/products/compass)

- **TablePlus** - Database GUI for multiple databases
  - [Website](https://tableplus.com/)

- **Redis Desktop Manager** - GUI for Redis
  - [GitHub](https://github.com/RedisInsight/RedisInsight)

## Debugging

- **Node Inspector** - V8 debugger
  - [GitHub](https://github.com/node-inspector/node-inspector)

- **Chrome DevTools** - Built-in debugging with `--inspect`
  - Run: `node --inspect server.js`

- **Prettier** - Code formatter
