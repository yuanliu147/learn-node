# Videos & Courses

Free and paid video resources for learning Node.js.

## Free Video Courses

- [Node.js Official Documentation Tutorials](https://nodejs.org/en/docs/guides/) - Official getting started guides
- [Traversy Media - Node.js Crash Course](https://www.youtube.com/watch?v=fBNz5xF-Kx4) - YouTube crash course (1.5 hours)
- [Academind - Node.js Complete Guide](https://www.youtube.com/watch?v=IPC7vGCY3CA) - Comprehensive YouTube tutorial
- [Fun Fun Function - Async JavaScript](https://www.youtube.com/watch?v=POk_kM0r8S8) - YouTube series on async patterns

## Paid Courses

- [Udemy - The Complete Node.js Developer Course](https://www.udemy.com/course/the-complete-nodejs-developer-course-2/) - Andrew Mead's comprehensive course
- [Pluralsight - Node.js: Getting Started](https://www.pluralsight.com/courses/node-js-getting-started) - Core fundamentals
- [Frontend Masters - Complete Node.js Developer](https://frontendmasters.com/courses/nodejs/) - In-depth backend development
- [Coursera - Server-side Development with NodeJS](https://www.coursera.org/learn/server-side-nodejs) - University of Hong Kong course

## YouTube Channels

- [Traversy Media](https://www.youtube.com/user/TechGuysTraining) - Web development tutorials
- [Academind](https://www.youtube.com/c/Academind) - JavaScript and Node.js tutorials
- [The Net Ninja](https://www.youtube.com/c/TheNetNinja) - Free Node.js and Express tutorials
- [Fireship](https://www.youtube.com/c/Fireship) - Quick, high-quality code tutorials

## Interactive Learning

- [Node School](https://nodeschool.io/) - Open source workshops that run in your terminal
- [Exercism.io - Node.js Track](https://exercism.io/tracks/javascript) - Practice problems with mentoring
- [Codewars - Node.js Challenges](https://www.codewars.com/kata/search/node) - Kata-style coding challenges
