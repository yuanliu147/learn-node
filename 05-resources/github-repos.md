# GitHub Repositories

Useful open-source Node.js repositories and projects to study.

## Official Node.js

- [nodejs/node](https://github.com/nodejs/node) - Official Node.js source code
- [nodejs/node.githubservices](https://github.com/nodejs/githubservices) - GitHub services integration
- [nodejs/node-v8.x](https://github.com/nodejs/node-v8.x) - Node.js v8 branch

## Learning Resources

- [nodeschool/workshoppers](https://github.com/nodeschool/workshoppers) - Interactive Node.js tutorials
- [substack/learnyounode](https://github.com/nodeschool/learnyounode) - Learn Node.js command-line workshop
- [wesbos/JavaScript30](https://github.com/wesbos/JavaScript30) - 30-day JavaScript coding challenge

## Express.js & Frameworks

- [expressjs/express](https://github.com/expressjs/express) - Fast, unopinionated web framework
- [expressjs/generator](https://github.com/expressjs/generator) - Express application generator
- [nestjs/nest](https://github.com/nestjs/nest) - Framework for building server-side applications
- [fastify/fastify](https://github.com/fastify/fastify) - Fast and low overhead web framework

## Tools & Utilities

- [npm/cli](https://github.com/npm/cli) - Official npm CLI repository
- [yarnpkg/yarn](https://github.com/yarnpkg/yarn) - Fast, reliable package manager
- [pm2/pm2](https://github.com/Unitech/pm2) - Advanced Node.js process manager
- [nodemon/nodemon](https://github.com/nodemon/nodemon) - Monitor for changes and auto-restart

## Testing Libraries

- [jestjs/jest](https://github.com/jestjs/jest) - Delightful JavaScript testing framework
- [mochajs/mocha](https://github.com/mochajs/mocha) - Simple, flexible JavaScript test framework
- [chaijs/chai](https://github.com/chaijs/chai) - BDD/TDD assertion library
- [supertest/supertest](https://github.com/visionmedia/supertest) - HTTP assertions made easy

## Boilerplates & Starters

- [sindresorhus/commander.js](https://github.com/sindresorhus/commander.js) - CLI utilities
- [SBoudrias/Inquirer.js](https://github.com/SBoudrias/Inquirer.js) - Interactive CLI prompts
- [chalk/chalk](https://github.com/chalk/chalk) - Terminal string styling
- [debug-js/debug](https://github.com/debug-js/debug) - Debugging utility

## Real-World Examples

- [freeCodeCamp/palmeiras-peladas-api](https://github.com/freeCodeCamp/palmeiras-peladas-api) - Example REST API
- [hashnode/starter-kit](https://github.com/hashnode/starter-kit) - Blogging platform starter
