| [Phase 1](phase-01-native-node.md) | 原生 Node.js 基础 | 2-3 周 | L2-L3 | 6 |
| [Phase 2](phase-02-async-stream.md) | 异步与流 | 2-3 周 | L3-L4 | 10 |
| [Phase 3](phase-03-framework.md) | 框架与工程化 | 2-3 周 | L2-L3 | 4 |
| [Phase 4](phase-04-ai-backend.md) | AI 后端专项 | 2-3 周 | L3-L4 | 7 |
| [Phase 5](phase-05-performance.md) | 性能优化 | 2-3 周 | L4-L5 | 11 |
| [Phase 6](phase-06-advanced-source.md) | 进阶与源码 | 3-4 周 | L5 | 5 |

## 里程碑

| 里程碑 | 完成条件 | 对应阶段 |
|--------|----------|----------|
| 🌱 入门 | 完成 Phase 0-1，能手写 HTTP 服务 | Phase 0, Phase 1 |
| 📦 基础 | 完成 Phase 2，理解异步和流 | Phase 2 |
| 🏗️ 工程 | 完成 Phase 3，能构建生产级应用 | Phase 3 |
| 🤖 AI | 完成 Phase 4，能构建 AI 后端服务 | Phase 4 |
| ⚡ 性能 | 完成 Phase 5，能排查和优化性能问题 | Phase 5 |
| 🔬 源码 | 完成 Phase 6，能阅读 Node.js 源码 | Phase 6 |
